## 2024-05-14 - Optimize useMemo iteration in AnalyticsView
**Learning:** Found a specific React anti-pattern in `src/components/Views/AnalyticsView.jsx` where a `useMemo` block used five separate iterations (`filter()`, `reduce()`, `forEach()`) over the same `filteredPosts` array to calculate metrics. This chaining caused massive performance degradation and O(M*N) memory allocations for large datasets.
**Action:** Always combine operations that process the same array within `useMemo` hooks into a single `for...of` loop to achieve O(N) complexity and prevent memory thrashing. Avoid `substring` comparisons for `Date` parsing if timezones are involved, as `new Date()` is safer.
